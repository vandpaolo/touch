# Phase T4b — Editor tabs / multi-document model

- **Goal:** Keep multiple parts open and switch between them via editor tabs (the multi-doc slice of T14 + open-decision #5, pulled forward).

- **Max:** Drag-reorder tabs; recent-parts; split view.
